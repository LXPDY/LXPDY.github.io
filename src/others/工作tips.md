#  一些工作方法论

## 时间管理

明确目标【三问】

- 交付目标
- 交付时间
- 交付优先级

4D原则

- 做 *Do it now.*
- 存 *Do it later.*
- 转 *Delegate.*
- 扔 *Don't do it.*

大石头理论【要事第一】

- 保证最关键的20%输出，对低价值事情说“不”

自我管理

- 将任务切割为单个动作
- 设置自己的DDL
